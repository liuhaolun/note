#### 通过setAttr和getAttr使批量生成的button，其变量可沟通
通过`setattr(self, attr_name, entry)`可以把entry这个本地变量存储为shell里的变量，名称是`attr_name`，需要注意别重名，方法是`attr_name = label.strip().replace(' ', '_').replace(':', '').lower()`构造独特名称。
```
r_acce_entry = getattr(self, '(r)_acce_')
r_acce_value = r_acce_entry.get()
print(f"(R) Acce 的值是: {r_acce_value}")
```
第一句获得了“引用”，第二句解析为值，第三句显示。Attr是全局的。
