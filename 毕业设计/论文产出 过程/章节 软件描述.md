Figure 3.5 A screenshot of the control software's main user interface. The displayed data has been adjusted to simultaneously illustrate the display of four plots and does not represent actual experimental data.

The control software incorporates modules for serial communication, data logging and exportation, and real-time data display. Furthermore, it can load a pre-trained neural network to perform adjustment direction prediction, automated adjustment, and autonomous assembly.
The left panel contains control-related widgets. Motion of the Z, X, and Y motorized stages is controlled via DS102 controllers. Z-axis movement is performed in 10微米 increments. Following each Z-axis movement, a delay (10 ms) is introduced to mitigate impulse effects. Subsequently, the software asynchronously communicates with multiple serial ports and records current sensor readings. These readings include force information, stage displacement, and inductive micrometer displacement. Additionally, the system can track the goniometer position for the preceding 15 steps, which corresponds to the hole's attitude. The 'Explore' function offers two methods: vertical explore and attitude explore. In these modes, the motorized stage or goniometer moves continuously until an abrupt change in force sensor readings, indicative of two-point contact, is detected. This specific detection does not employ deep learning methods. The 'Learn' function also provides two methods. Firstly, the operator can assess the current relative pose and record the corresponding force information; this dataset is then exported for model improvement. Secondly, the operator can perform an action corresponding to the current labeled state; this dataset is utilized for imitation learning. (However, the effect of imitation learning is negligible in practice). The 'Hand' button toggles manual control mode. The 'Auto' function enables autonomous assembly. After two-point contact is detected and this function is activated, the software cyclically executes the procedures outlined in 【FIGURE】. Finally, the 'Status' function allows viewing of the currently active difficulty criteria.

The right-side menu concludes plots for data visualization. Data from the 3-axis force sensor (connected to the shaft's gripper) includes the Z-axial force F_z. This force is subject to a project limitation, requiring it not to exceed 2 N. The relationship between F_z-Z-displacement is displayed as a scatter plot, given that Z-displacement is measured in 10 μm increments. The bottom-left plot shows the X-axial and Y-axial forces from this 3-axis force sensor. During typical experiments, the absolute values of these forces rarely exceed 20 mN. The bottom-right plot displays the vector sum of readings from three Z-axis force sensors (connected to the hole's gripper) and the calculated current polar angle. These plots assist the operator in intepreting sensor readings. The data processing involved aims to simplify the presented information.
