# 非常具有责任心
我擅长确保交给我的任务能符合要求地完成，托付我转交给他人的任务，我能够仔细修改沟通内容、确保不发生歧义；当我向他人提出要求、或进行告知时，我能够确保附带所有所需信息。
# 具备高效沟通能力
在重要的沟通事项上，我有能力去除不必要的、无意义的沟通内容，专注于最关键、必须在交流中达成一致、并各自遵守的。
# 具有组织能力
我有能力合理地根据个人的情况不同进行任务分配，善于利用表格工具分解任务期限，我擅长在群聊中进行说明，并且避免队员看到大量“与他们个人无关的内容”刷屏。我能够把任务的进展及时通知到每个队员，且留出时间避免干扰各自进度。
# 擅长调动积极性
我不认同一些队员的能力更差，或者一部分人应该做更多的事情。我有意愿：作为队长，帮助它们找到教学资源来完成任务；通过合适的分组，让能够合作的队员共同完成任务。我反对“因为一些人的能力差，所以他们该听从分组中更强的人”，我认为“更有能力完成任务”，本质上是对任务有全局的观念，但没有人能在学习前了解全部细节。通过细致地分解任务（作为队长的工作），我能够确保每位队员只需要了解很小部分的细节，这样所有队员都有“能力”来学习自己的部分。最终，能够避免一部分消极应对。
# 具有很强的查找资料能力
我善于自行查询资料，并且有自己寻找资料的方法。通过整理资料和希望查找的信息，我能够减少遗漏。我了解互联网，能够避开垃圾信息，找到关键信息；我还擅长沟通，能够从他人处了解到在哪里可能获得什么资料。
